# 忆故思迁的无名小站
#### 这是一个使用了Vue,Vite,Element-Plus,Axios,ArcoDesign构建的个人首页
#### 作者是一个大二在学的学生，技术力和精力有限，正在努力学习中

## 运行方式：
### 一、使用Vite:下载-->解压-->直接拖入VsCode-->Vite自动提示，根据提示安装运行
### 二、手动安装包：下载-->解压-->使用IDE打开-->进入命令行-->cd进入文件夹-->npm install-->npm run dev
